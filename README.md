# Sheet Hive

Sheet Hive is a spreadsheet-driven data validation platform designed for
business-specific workflows.

## Purpose

This organization contains repositories related to **Sheet Hive**.

Currently, only the **web version** is publicly available as a
**portfolio / demonstration project**.
Other components (core logic, desktop app, plugins) are under active
development and are not publicly released at this time.

## Architecture Overview

Sheet Hive is designed as a multi-component system:

- Web: Browser-based UI and validation demo
- Core: Shared validation and rule execution logic
- Desktop: Desktop-oriented workflow integration
- Plugins: Case-specific validation extensions

For portfolio purposes, the web component is showcased first.
